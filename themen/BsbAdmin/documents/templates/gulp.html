Gulp is build system supported by pages, gulp's use of streams and code-over-configuration makes for a simpler and more intuitive build.


Installing Gulp

To install gulp first you must have NodeJs installed, NodeJS will have npm.

And then please follow these instrucations:

  1. Install gulp globally by running the following command npm install -g gulp
  2. In you downloaded package, navigate to gulp/ directory and copy both package.json , gulpfile.js to your project root.
  3. Navigate to the root directory of your project, then run npm install.

Commands

gulp build

This will automatically compile & minify your assets

gulp watch

This will automaically compile the pages Sass files on save

gulp compile-scss

This will compile the sass files


Troubleshooting

Should you encounter problems with installing dependencies or running Gulps commands, first delete the /node_modules/ directory generated by npm. Then, rerun npm install.
